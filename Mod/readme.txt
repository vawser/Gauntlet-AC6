# Installation
1. Download ModEngine2 for Armored Core VI. You can find the latest release at https://github.com/soulsmods/ModEngine2/releases. 
Follow its instructions for installation.
2. Extract the mod into the"mod" folder that ModEngine2 comes with.
3. Click "launchmod_armoredcore6.bat" to start the game with the mod enabled.

# Uninstallation
Restore the normal "regulation.bin" you backed up. 

If you did not backup the file, delete the modded "regulation.bin" and use the "Verify integrity of game files" action in Steam to restore the original.
